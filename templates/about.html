Our Technology

Powered by Deep Learning

CNN Architecture

Multi-layer convolutional neural network with 4 conv blocks, batch normalization, and dropout layers

Dataset

Trained on 50,000+ dermatology images across 24 different skin conditions and diseases

TensorFlow & Keras

Built using industry-standard deep learning frameworks for reliable and accurate predictions

Data Augmentation

Advanced preprocessing with rotation, flip, zoom, and brightness adjustments for robustness

Technical Details

Model Architecture

Input Layer

150x150x3

Conv Blocks

4 Blocks (32-64-128-256)

Dense Layers

512 → 256 neurons

Output Layer

24 Classes (Softmax)

Training Details
  • Optimizer: Adam (lr=0.0001)
  • Loss Function: Categorical Crossentropy
  • Batch Size: 16
  • Epochs: 50 (with early stopping)
Regularization
  • Dropout: 0.25 - 0.5
  • Batch Normalization: Yes
  • L2 Regularization: 0.001
  • Data Augmentation: Extensive
